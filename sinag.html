Sinag - Fighting Game

Summary

Currently in development under Ranida Games that will release soon.

A casual mobile fighting game for beginners but with deep game mechanics to appeal for competitive gamers. The game will feature characters and backgrounds based on anything that represents the Filipino culture.

What I did

  • Development for editor extension of Universal Fighting Engine 2 for project specifications.
  • UI development.
  • Implementation of Advertisements (AppLovin MAX) and In-App-Purchase
